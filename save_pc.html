Step 6: Where to put your TTS file on a PC.
Take the downloaded file to this file path. This is the where TTS will be able to read it. Disclaimer: Your my games folder might not be in the documents folder under onedrive. It could be under your regular documents folder depending on where steam decides to put it.You can also find a tokens file on the steam workshop so you can have something representing them. Any other files that you have added from anyother location or computer will also appear here.
